═══════════════════════════════════════════════════════════════════════════════
🎯 FLUJO DE DATOS EN FRONTEND
═══════════════════════════════════════════════════════════════════════════════

User Navegates to /drivers
        ↓
Component Mounts (DriversPage)
        ↓
useF1Data hook ejecuta
        ↓
fetch('/api/f1/drivers')
        ↓
Next.js Route Handler
        ↓
f1Service.fetchF1Data('drivers')
        ↓
Check MongoDB Cache
        ├─ Valid? → Return data
        └─ Expired? → Fetch Open F1 API
        ↓
Update state (data, loading, error)
        ↓
Re-render con datos
        ↓
Map array y renderizar DriverCards
        ↓
Mostrar Grid con 20-30 pilotos


═══════════════════════════════════════════════════════════════════════════════
💡 EJEMPLOS DE USO
═══════════════════════════════════════════════════════════════════════════════

1. USAR HOOK EN COMPONENTE

import { useF1Data } from '@/lib/hooks/useF1Data';

export function MyComponent() {
  const { data, loading, error } = useF1Data({
    endpoint: 'drivers',
  });

  if (loading) return <LoadingSpinner />;
  if (error) return <ErrorMessage message={error.message} />;

  return (
    <div>
      {Array.isArray(data) && data.map(driver => (
        <DriverCard key={driver.driver_number} {...driver} />
      ))}
    </div>
  );
}


2. USAR FORMATTERS

import { formatDate, getCountryFlag } from '@/lib/utils/formatters';

export function MyCard() {
  return (
    <div>
